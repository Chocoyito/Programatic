Los Operadores

Los operadores son una herramienta muy importante en la programación. Nos sirven para manejo y proceso de datos dentro de un programa o aplicación. Existen tres tipos de operadores: operadores aritméticos, operadores relacionales y operadores lógicos. A continuación se explicarán las bases de los distintos tipos de operadores.


Operadores Aritméticos

A como su nombre hace referencia, los operadores aritméticos funcionan igual como lo hacen en la matemática o en una calculadora. Los operadores aritméticos dentro del lenguaje de programación Java y estándar en otros lenguajes son los siguientes:

Operador Símbolo Uso Símbolo Alternativo Uso
Suma + Suma dos valores ++ Suma 1 a un valor
Resta - Resta dos valores -- Resta 1 a un valor
Multiplicacion * Multiplica dos valores
división / Divide dos valores
Modulo % Devuelve el resto de la división de dos valores


Sintaxis

Dentro de un lenguaje de programación una operación entre dos elementos o variables produce una respuesta. La sintaxis para una operación simplemente consiste en un número o variable seguido por el símbolo de la operación y luego el otro número o variable. Es importante mencionar que para almacenar la respuesta se debe igualar a una variable del mismo tipo de dato.


Operadores Relacionales

Los operadores relacionales funcionan a partir de la comparación de dos elementos. La respuesta que se obtiene a través de la comparación a través de un operador relacional es de tipo boolean. Los operadores relacionales dentro del lenguaje de programación Java y estándar en otros lenguajes son los siguientes:

Operador Símbolo Uso
Igual a == Verifica si ambos valores son iguales
Diferente de != Verifica si ambos valores son diferentes
Mayor que > Verifica si el primer valor es mayor que el segundo
Menor que < Verifica si el primer valor es menor que el segundo
Mayor o igual que >= Verifica si el primer valor es mayor o igual que el segundo
Menor o igual que <= Verifica si el primer valor es menor o igual que el segundo


Sintaxis

Dentro de un lenguaje de programación una comparación entre dos elementos o variables produce una respuesta de tipo booleana como se mencionó anteriormente. La sintaxis para una operación relacional consiste en un número o variable seguido por el símbolo de la operación y luego el otro número o variable. Esta devolverá un valor de tipo booleano el cual se deberá guardar dentro de una variable.


Operadores Lógicos

Los operadores lógicos funcionan a partir de la comparación de dos elementos booleanos. La respuesta que se obtiene a través de la comparación a través de un operador lógico es de tipo boolean. Los operadores relacionales dentro del lenguaje de programación Java y estándar en otros lenguajes son los siguientes:

Operador Símbolo Uso
Y && El primer valor Y el segundo valor deben de ser Verdaderos para devolver un valor Verdadero
O || Al menos uno de los dos valores debe de ser Verdadero para devolver un valor Verdadero


Sintaxis

Dentro de un lenguaje de programación una comparación entre dos elementos o variables produce una respuesta de tipo booleana como se mencionó anteriormente. La sintaxis para una operación lógica consiste en un variable de tipo booleana seguido por el símbolo de la operación y luego la otra variable tipo booleana. Esta devolverá un valor de tipo booleano el cual se deberá guardar dentro de una variable.
