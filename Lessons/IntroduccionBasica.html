Introducción

La programación se puede considerar un rompecabezas con soluciones infinitas. Estas soluciones las podemos usar para resolver cualquier problema que se nos ocurra. Las piezas del rompecabezas, por otro lado, son los elementos más básicos de la programación los cuales se explicaran a través estas lecciones. Estos elementos consisten en: Las variables, Los operadores, las condicionales y los bucles. A través de estas herramientas es posible la creación de cualquier tipo de Software básico.


Las Variables

Para ponerlo en términos simples, las variables consisten en un espacio en memoria con un nombre que el programador define dónde se guarda un tipo específico de dato.

Existen tres tipos de variables:

  • Primitivas ie. int, char, boolean
  • Estructuradas ie. Array, String
