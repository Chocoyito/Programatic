Las Variables

Las variables son las herramientas más importantes de la programación. Estas nos sirven para el almacenamiento y manejo de datos dentro de un programa o aplicación. Dentro de lo que son las variables, existen diferentes tipos de este: de números, de sí o no , de caracteres y de listas. A continuación se explicarán las bases de los distintos tipos de variables y cómo usarlas.


Las de números

Dentro de las variables que tienen que ver con números se encuentran aquellas mencionadas anteriormente como int, float y double. Además de estas variables, existen otras variables como byte, short y long.

Las variables de números se pueden dividir en dos categorías, en números enteros y decimales. Las variables que caen dentro de los números enteros son: byte, short, int y long. La diferencia entre estas variables recae en la cantidad de dígitos pueden contener. Por lo general se usa la variable int para guardar números enteros gracias a su flexibilidad y optimización.

Nombre Tamaño Rango
byte 8 bit -128 a 128
short 16 bits -32,768 a 32,768
int 32 bit -2,147,483,648 a 2,147,483,648
long 64 bits -9*10^18 a 9*10^18

Por el otro lado, las variables que caen dentro de los decimales son: float y double. Estas dos variables también se diferencian por la cantidad de dígitos que pueden almacenar. En la programación se usan ambas variables dependiendo del contexto y la necesidad.

Nombre Tamaño Rango
float 32 bit -3.4*10^38 a 3.4*10^38
double 64 bits -1.79*10^308 a 1.79*10^308


Sintaxis

La sintaxis para la creación de una nueva variable consiste en escribir el tipo de variable seguido por un nombre (cerrado por un punto y coma).

El guardado de números enteros dentro de una variable creada consiste en simplemente igualar la variable al número entero a guardar. Cualquier tipo de dato fuera de un número entero causaría un error de compilación.


La de Si o No

La variable de "Si o No" se conoce como bool. Esta es una variable con el tamaño de un bit y el rango de 0 y 1. Todo esto queriendo decir que la variable bool es una variable binaria que permite guardar un valor de verdadero y falso.

Nombre Tamaño Rango
boolean 1 bit True o False


Sintaxis

La sintaxis para la creación de una nueva variable de tipo bool consiste en escribir su nombre y el nombre de la variable (cerrado por un punto y coma). Después de esto se iguala la variable a True (Verdadero) o False (Falso). Por otro lado, uno es capaz de asignar los valores a través de un 0 (Falso) o 1 (Verdadero).


La de Caracteres

La variable usada para guardar un carácter se le conoce como char (viniendo de la palabra character). Esta variable es capaz de guardar solamente un carácter Unicode, Unicode siendo el estándar de codificación de caracteres.

Nombre Tamaño Rango
char 16 bit Unicode


Sintaxis

La sintaxis para la creación de una nueva variable de tipo char solamente consiste en escribir su nombre y el nombre de la variable (cerrado por un punto y coma). Después de esto se iguala la variable al carácter que se le asigna dentro de comillas.


Las de Listas

Existen dos tipos de variables que se pueden considerar variables de listas. Estas variables consisten en String y Arrays. La variable String se usa para guardar una lista de caracteres donde es capaz de guardar caracteres Unicode. Por el otro lado están los Arrays. Los arrays no es una variable de por sí pero un modificador que se le Se puede agregar a otro tipo de variable para hacer una lista de valores de su tipo.

Nombre Tamaño Rango
String 1 bit + x bits Array de Caracteres
Array 1 bit + x bits Lista de Variables

Los x bits representan la cantidad de bits de todos los elementos dentro del Array.

(Un String de 10 caracteres tendría el tamaño máximo de 160 bits más 1 bit del String en sí)


Sintaxis

La sintaxis para crear una variable String consiste en escribir la variable seguido por un nombre. Después de esto se iguala la variable al texto que se le asignará dentro de comillas (y se termina cerrando con un punto y coma).

En el caso de un Array primero se declara el tipo de variable del que va a ser el Array junto a corchetes( [ ] ). Luego de eso se escribe el nombre del array y se iguala a los valores que va a contener. El formato para guardar en un array es dentro de llaves ( {} ) se escriben los valores (respetando su sintaxis individual) separados por comas.
