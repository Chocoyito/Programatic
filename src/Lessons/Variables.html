Las Variables

Las variables son las herramientas mas importantes de la programacion. Estas nos sirven para el almacenamiento y manejo de datos dentro de un programa o aplicacion. Dentro de lo que son las variables, existen diferentes tipos de este: de numeros, de si o no , de caracteres y de listas. A continuacion se explicaran las bases de los distintos tipos de variables y como usarlas.


Las de numeros

Dentro de las variables que tienen que ver con numeros se encuentran aquellas mencionadas anteriormente como int, float y double. Ademas de estas variables, existen otras variables como byte, short y long.

Las variables de numeros se pueden dividir en dos categorias, en numeros enteros y decimales. Las variables que caen dentro de los numeros enteros son: byte, short, int y long. La diferencia entre estas variables recae en la cantidad de digitos pueden contener. Por lo general se usa la variable int para guardar numeros enteros gracias a su flexibilidad y optimizacion.

Por el otro lado, las variables que caen dentro de los decimales son: float y double. Estas dos variables tambien se diferencian por la cantidad de digitos que pueden almacenar. En la programacion se usa ambas variables dependiendo del contexto y la necesidad.

Sintaxis

La sintaxis para la creacion de una nueva variable consiste en escribir el tipo de variable seguido por un nombre (cerrado por un punto y coma).

El guardado de numeros enteros dentro de una variable creada consiste en simplemente igualar la variable al numero entero a guardar. Cualquier tipo de dato afuera de un numero entero causaria un error de compilacion.

Sintaxis

La sintaxis para la creacion de una nueva variable de tipo bool consiste en escribir su nombre y el nombre de la variable (cerrado por un punto y coma). Despues de esto se iguala la variable a True (Verdadero) o False (Falso). Por otro lado, uno es capaz de asignar los valos a traves de un 0 (Falso) o 1 (Verdadero).
