Introduccion

La programacion se puede considerar un rompecabezas con soluciones infitas. Estas soluciones las podemos usar para resolver cualquier problema que se nos ocurra. Las piezas del rompecabezas, por otro lado, son los elementos mas basicos de la programacion los cuales se explicaran a traves estas lecciones. Estos elementos consisten en: Las variables, Los operadores, las condicionales y los bucles. A traves de estas herramientas es posible la creacion de cualqueir tipo de Software basico.


Las Variables

Para ponerlo en terminos simples, las variables consisten en un espacio en memoria con un nombre que el programador define donde se guarda un tipo especifico de dato.

Existen tres tipos de variables:

  • Primitivas ie. int, char, boolean
  • Estructuradas ie. Array, String
